# 販売管理・見積管理システム 設計書

## 会社概要（井上電設株式会社）
- 産業用集塵装置・空気輸送装置の製造・販売
- 製品タイプ: BFQ（バグフィルタ）, BFR（逆洗式バグフィルタ）, SCA（定量排出装置）
- 見積は製品本体＋オプション（RV・シェーカー・制御盤等）の積み上げ構造

## システム構成

```
frontend/   React + TypeScript + Vite
backend/    Python FastAPI
database/   PostgreSQL
```

## データベース設計

### マスタ系
- customers（顧客マスタ）
- products（商品・製品マスタ）
- product_options（製品オプションマスタ）
- suppliers（仕入先マスタ）
- users（ユーザーマスタ）

### 見積管理
- quotations（見積ヘッダ）
- quotation_items（見積明細）
- quotation_options（見積オプション明細）

### 受注・販売管理
- orders（受注ヘッダ）
- order_items（受注明細）
- invoices（請求書）

### 在庫・仕入管理
- inventory（在庫）
- purchase_orders（発注書）
- purchase_order_items（発注明細）
- stock_movements（入出庫履歴）
